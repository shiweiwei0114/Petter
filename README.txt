II.		FEATURES OF THE WEBSITE

1). RETRIEVE (SEARCH)
	The user can search a pet by choosing Pet-type, Sex, State and type in City on the Home page.

2).	INSERT (ADD)
	The user can click "Add A Pet" button in the Home page, it will lead the user to another page to add a pet.
	-- If the user adds a valid pet, it will show "Insert Successfully" and show the Pet ID.
	-- If the user adds a invalid pet, it will show "Insert Unsuccessful" and display the necessary items missing.
	
3). UPDATE
	The user can click "Update Pet" button in the Home page, it will lead the user to another page to update a pet.
	-- Pet ID is required to update the pet information. 
	-- If the user updates the pet with valid information, it will show "Update Successful".
	-- ERROR: If the user enters an invalid pet ID, the pages still shows "Update Successful".

4).	DELETE
	The user can click "Delete Pet" button in the Home page, it will lead the user to another page to delete a pet.
	-- Pet ID is required to update the pet information.
	-- ERROR: If the user enters an invalid pet ID, the pages still shows "Delete Successful".
